# OpenNetBattle Server – Soft Scaling Encounters

This server implements **player-aware encounters** that scale enemy difficulty
based on player max HP (mods allowed) instead of enforcing stat limits.

## What’s included
- Player-aware encounter selection (ezencounters patch)
- Tiered encounter tables (rank scaling V1–V4)
- Optional reward scaling

## Not included
- Copyrighted or third-party asset packs (bring your own)
- Player save/memory data

## Requirements
- OpenNetBattle
- ezlibs
- ezlibs-assets (not included)
